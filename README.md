# Plant Watering system

## Dev

- load db shema
- create .env with `DATABASE_URL=`

```sh
yarn kysley-codegen
// copy generated code from ./node_modules/kysely-codegen/dist/db.d.ts
```

- start frontend

```sh
yarn
yarn dev
```
